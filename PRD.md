# Align. — Product Requirements Document
## Full Functionality Build · Next.js 14 PWA · Supabase + Dexie Offline-First

## 1. PRODUCT DEFINITION

Align. is a 14-day commitment engine. Not a task manager. Not a planner.

The core constraint: **14 days. 3 moves per day. nothing more.**

The philosophy: most productivity failure is not lack of tools — it is too much optionality. Align. removes optionality by design.

### Core loop
```
Set 1-3 directions for the window
  ↓
Each day: pick ≤3 moves (concrete, doable in one sitting)
  ↓
Each night: did you show up? yes / avoided
  ↓
Day 14: window closes. reflect. open a new one.
```

### What this is NOT
- Not a calendar, habit tracker, project manager, or notes app
- Not social, not AI-assisted, not gamified

### 6.3 Sync Engine (lib/db/sync.ts)

Strategy: local-first writes → background push to Supabase → pull on app load.

Conflict resolution:
- `_synced=0` (unsynced local): LOCAL WINS — skip on pull
- `_synced=1` (synced): SERVER WINS on re-pull
- Boolean progress flags (dropped, promoted, status='closed'): true/closed never reverts

## 7. AUTH

### Complete flow
```
Any protected route → AuthGuard (server component)
  → no session → redirect /auth instantly (no flash)

/auth: email → supabase.auth.signInWithOtp({
  email, options: { emailRedirectTo: origin + '/auth/callback' }
})
→ show "Check your inbox." state

/auth/callback?code=xxx:
  → exchange code for session
  → check if profile exists in Supabase
  → no profile → /onboarding
  → profile exists → /home

/home: AuthGuard passes, AppContext loads, WindowClosedFlow check runs
```

### /auth/callback/route.ts
```typescript
import { createRouteHandlerClient } from '@supabase/auth-helpers-nextjs'
import { cookies } from 'next/headers'
import { NextResponse } from 'next/server'

export async function GET(request: Request) {
  const { searchParams, origin } = new URL(request.url)
  const code = searchParams.get('code')
  if (code) {
    const supabase = createRouteHandlerClient({ cookies })
    const { error } = await supabase.auth.exchangeCodeForSession(code)
    if (!error) {
      const { data: { user } } = await supabase.auth.getUser()
      if (user) {
        const { data: profile } = await supabase
          .from('profiles').select('id').eq('id', user.id).single()
        if (!profile) return NextResponse.redirect(`${origin}/onboarding`)
      }
    }
  }
  return NextResponse.redirect(`${origin}/home`)
}
```

### Sign out
```typescript
async function signOut(router: AppRouterInstance) {
```

### persistOnboarding
```typescript
async function persistOnboarding(state: OnboardingState, userId: string) {
  const now = new Date().toISOString()
  const today = format(new Date(), 'yyyy-MM-dd')
  const cycleId = newId()
  const colorMap: Array<'terra'|'forest'|'slate'> = ['terra', 'forest', 'slate']
  const validDirs = state.directions.filter(d => d.trim())

  await db.profiles.put({
    id: userId, name: state.name.trim(),
    age: state.age ? parseInt(state.age, 10) : undefined,
    timezone: Intl.DateTimeFormat().resolvedOptions().timeZone,
    notifEnabled: state.notifEnabled,
    notifMorningTime: '08:00', notifNightTime: '21:30',
    pushSubscription: localStorage.getItem('align_push_sub') ?? undefined,
  })

  await db.cycles.put({
    id: cycleId, userId, startDate: today,
    endDate: format(addDays(new Date(), state.cycleLength - 1), 'yyyy-MM-dd'),
    lengthDays: state.cycleLength, status: 'active', createdAt: now, _synced: 0,
  })

  for (const [i, title] of validDirs.entries()) {
    await db.directions.put({
      id: newId(), cycleId, userId, title: title.trim(),
      color: colorMap[i], position: (i + 1) as 1|2|3, createdAt: now, _synced: 0,
    })
  }

  localStorage.setItem('align_onboarded', 'true')
  syncAll(userId).catch(() => {})  // fire and forget
}
```

### Push registration (lib/notifications/push.ts)
```typescript
export function urlBase64ToUint8Array(base64: string): Uint8Array {
  const padding = '='.repeat((4 - base64.length % 4) % 4)
  const b64 = (base64 + padding).replace(/-/g, '+').replace(/_/g, '/')
  return new Uint8Array([...atob(b64)].map(c => c.charCodeAt(0)))
}

export async function registerPushSubscription(userId: string): Promise<boolean> {
  if (!('Notification' in window) || !('serviceWorker' in navigator)) return false
  const perm = await Notification.requestPermission()
  if (perm !== 'granted') return false
  try {
    const reg = await navigator.serviceWorker.ready
    const sub = await (await reg.pushManager.getSubscription())
      ?? await reg.pushManager.subscribe({
        userVisibleOnly: true,
        applicationServerKey: urlBase64ToUint8Array(process.env.NEXT_PUBLIC_VAPID_PUBLIC_KEY!)
      })
    const json = JSON.stringify(sub)
    localStorage.setItem('align_push_sub', json)
    await db.profiles.update(userId, { pushSubscription: json, notifEnabled: true })
    return true
  } catch { return false }
}
```

```javascript
self.addEventListener('push', event => {
  const data = event.data?.json() ?? {}
  event.waitUntil(
    self.registration.showNotification(data.title ?? 'Align.', {
      body: data.body, icon: '/icons/icon-192.png',
      data: { url: data.url ?? '/home' }, vibrate: [100,50,100],
    })
  )
})
self.addEventListener('notificationclick', event => {
  event.notification.close()
  event.waitUntil(
    clients.matchAll({ type:'window' }).then(list => {
      for (const c of list) if (c.url.includes('/home') && 'focus' in c) return c.focus()
      return clients.openWindow(event.notification.data.url)
    })
  )
})
```
